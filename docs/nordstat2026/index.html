The parametric bottleneck

This is important and this is a definition.

\[p(y \mid \theta) = \sum_{k=1}^{K} \pi_k \, f(y \mid \mu_k, \sigma_k^2)\]

Problems:

  • \(K\) must be fixed before seeing data
  • Model selection over \(K\) is awkward
  • True \(K\) may be unknown or infinite

Idea: put a prior directly over the space of distributions — let the data determine complexity.

The Dirichlet Process

Definition (Ferguson 1973). A random probability measure \(G\) on \((\Theta, \mathcal{B})\) is a Dirichlet Process with base measure \(G_0\) and concentration \(\alpha > 0\), written

\[G \sim \mathrm{DP}(\alpha, G_0),\]

if for every finite measurable partition \(\{B_1, \ldots, B_k\}\) of \(\Theta\),

\[(G(B_1), \ldots, G(B_k)) \sim \mathrm{Dir}(\alpha G_0(B_1), \ldots, \alpha G_0(B_k)).\]

Key properties:

  • \(\mathbb{E}[G(B)] = G_0(B)\) — the base measure is the prior mean
  • \(\mathrm{Var}[G(B)] = \frac{G_0(B)(1-G_0(B))}{\alpha + 1}\) — \(\alpha\) controls concentration
  • Realizations are discrete with probability one

Stick-breaking representation

Sethuraman (1994): \(G \sim \mathrm{DP}(\alpha, G_0)\) if and only if

\[G = \sum_{k=1}^{\infty} w_k \, \delta_{\theta_k}\]

where \(\theta_k \overset{\text{iid}}{\sim} G_0\) and

\[w_k = v_k \prod_{j < k} (1 - v_j), \quad v_k \overset{\text{iid}}{\sim} \mathrm{Beta}(1, \alpha).\]

The weights \((w_k)\) satisfy \(\sum_{k=1}^\infty w_k = 1\) a.s.

Interpretation of \(\alpha\): small \(\alpha \Rightarrow\) few dominant atoms; large \(\alpha \Rightarrow\) many small atoms, \(G \approx G_0\).

Chinese Restaurant Process

The DP induces a clustering distribution on observations. If \(\theta_1, \ldots, \theta_{n-1} \sim G\) have been drawn, then:

\[\theta_n \mid \theta_1, \ldots, \theta_{n-1} \sim \frac{\alpha}{\alpha + n - 1} G_0 + \frac{1}{\alpha + n - 1} \sum_{i=1}^{n-1} \delta_{\theta_i}\]

This is the Pólya urn scheme. Cluster structure: customer \(n\) joins

Event Probability
Table \(k\) (already occupied by \(n_k\) customers) \(n_k / (\alpha + n - 1)\)
New table \(\alpha / (\alpha + n - 1)\)

Expected number of clusters: \(\mathbb{E}[K_n] \approx \alpha \log n\).

DP Mixture Model

Combine the DP prior with a likelihood to get a flexible density estimator:

\[\begin{aligned} G &\sim \mathrm{DP}(\alpha, G_0) \\ \phi_i \mid G &\overset{\text{iid}}{\sim} G \\ y_i \mid \phi_i &\sim f(y \mid \phi_i) \end{aligned}\]

Marginalizing over \(G\) gives a random partition model. With \(G_0 = \mathcal{N}(\mu_0, \kappa_0^{-1})\) and \(f(y \mid \phi) = \mathcal{N}(y \mid \phi, \sigma^2)\) this is a Gaussian DP mixture.

The posterior \(G \mid y_1, \ldots, y_n\) is again a DP:

\[G \mid y_{1:n} \sim \mathrm{DP}\!\left(\alpha + n,\; \frac{\alpha}{\alpha+n} G_0 + \frac{1}{\alpha+n}\sum_{i=1}^n \delta_{y_i}\right)\]

Gibbs Sampler for DP Mixtures

Collapsed sampler (Neal 2000, Algorithm 3). Let \(z_i \in \{1,\ldots,K\}\) be the cluster label for observation \(i\).

For each \(i = 1, \ldots, n\):

  1. Remove \(i\) from its cluster; update counts \(n_{-i,k}\)
  2. Sample new label from:

\[p(z_i = k \mid z_{-i}, y) \propto \begin{cases} n_{-i,k} \cdot p(y_i \mid y_{-i, k}) & k \text{ existing} \\[4pt] \alpha \cdot p(y_i \mid G_0) & k \text{ new} \end{cases}\]

where \(p(y_i \mid y_{-i,k})\) is the posterior predictive under \(G_0\).

For conjugate \(G_0\) this is available in closed form — no MH step needed.

Summary

What we covered:

  • DP is a prior over distributions with two parameters: \(\alpha\) (concentration) and \(G_0\) (base measure)
  • Stick-breaking gives a constructive representation
  • CRP describes the induced clustering
  • DP mixtures are flexible density estimators
  • Conjugate Gibbs sampler is clean and efficient

Key formulas:

Object Expression
Prior mean \(\mathbb{E}[G] = G_0\)
CRP new table \(\alpha/(\alpha+n-1)\)
Posterior \(\mathrm{DP}(\alpha+n, \tilde{G})\)
\(\mathbb{E}[K_n]\) \(\approx \alpha \log n\)
